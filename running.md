# Running this project

## Requirements

- Python 3.12+ 
- `pytest` to run the tests

## Running the server

```sh
./script/server
```

Serves on `http://localhost:8080` by default. To use a different port:

```sh
./script/server --port 9000
```

Stop it with Ctrl-C.

## Running the tests

```sh
./script/test
```
