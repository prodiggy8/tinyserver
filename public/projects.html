Raw-socket HTTP/1.1 server

A from-scratch HTTP/1.1 server in Python, built directly on raw TCP sockets — no http.server, no third-party web framework. Handles request parsing, chunked transfer encoding, keep-alive connections, routing, and static file serving. This page is served by it.
